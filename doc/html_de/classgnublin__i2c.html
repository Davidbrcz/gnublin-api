int gnublin_i2c::receive ( unsigned char *  RxBuf,
int  length 
)

Empfängt Bytes vom I2C Bus.

Diese Funktion liest "length" Anzahl an Bytes vom I2C Bus und speichert diese in "RxBuf". Bei Erfolg wird 1 zurück gegeben, bei Misserfolg -1.
Beispiele:
Lese 2 Bytes und speichere diese in "buf":
receive(buf, 2);

Parameter:
RxBufEmpfangs Puffer. Die gelesenen Bytes werden hier gespeichert.
lengthAnzahl der zu lesenden Bytes.
Rückgabe:
Erfolg: 1, Misserfolg: -1
